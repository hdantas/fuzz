source code 

Primitive that cycles through a library of "bad" strings. The class variable 'fuzz_library' contains a list of smart fuzz values global across all instances. The 'this_library' variable contains fuzz values specific to the instantiated primitive. This allows us to avoid copying the near ~70MB fuzz_library data structure across each instantiated primitive.

Parameters:
  • value (String) - Default string value
  • size (Integer) - (Optional, def=-1) Static size of this field, leave -1 for dynamic.
  • padding (Character) - (Optional, def="\x00") Value to use as padding to fill static field size.
  • encoding (String) - (Optonal, def="ascii") String encoding, ex: utf_16_le for Microsoft Unicode.
  • fuzzable (Boolean) - (Optional, def=True) Enable/disable fuzzing of this primitive
  • max_len (Integer) - (Optional, def=0) Maximum string length
  • name (String) - (Optional, def=None) Specifying a name gives you direct access to a primitive
